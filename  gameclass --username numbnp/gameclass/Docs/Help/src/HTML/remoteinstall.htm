Удаленная инсталляция и обновление клиентов GameClass

Если на клиентском компьютере установлен клиент меньшей версии, чем сервер, то он будет автоматически обновлен до новой версии.
Для этого в настройках системы на закладке "GC3 клиент" нужно в группе "Автоматическая установка" указать путь к инсталляционному файлу GameClass (обычно это gc3setup.exe).

Если Вы хотите чтобы обновление осуществлялось автоматически (при указании правильного пути и несовпадении версии сервера и клиента), то еще установите галочку.

Если указан только путь, то обновлять клиентов можно под оператором при наличии им права "Деинсталляция/закрытие клиентов" при помощи пункта меню Менеджер->Управление GC3 клиентами.

Все это работает ТОЛЬКО начиная с версии 3.84 beta 3, т.е. установив ее Вы сможете обновить только следующую версию.
