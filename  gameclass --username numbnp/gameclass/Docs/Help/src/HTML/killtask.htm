Подробнее о возможностях системы

Удаленное завершение процессов.

Для завершения процессов компьютера выберите его в списке компьютеров и вызовите окно завершения задач одним из следующих способов:

  • клавиша Ctrl+Del
  • через контекстное меню (правой кнопкой на выбранном компьютере)
  • меню "Действия/Дополнительные функции/Завершить задачи"

В левой части окна завершения задач расположен список текущих процессов. Из этого списка исключены некоторые стандартные процессы Windows (но не все).

Посмотреть, как выглядит окно завершения задач




В правой части окна завершения задач расположен список шаблона задач для завершения. Наполнив шаблон типичными процессами Вашего интернет салона (Игры и другие приложения), вы можете использовать функцию "Снятие задач по шаблону", ее можно вызвать одним из следующих способов:

  • клавиша Shift+Del
  • через контекстное меню (правой кнопкой на выбранном компьютере)
  • меню "Действия/Дополнительные функции/Завершить задачи по шаблону"

Заполненный шаблон может использоваться в функции "Снять задачи после завершения работы посетителя после N минут" (функцию включает менеджер). Применение этой функции позволит АВТОМАТИЗИРОВАТЬ закрытие всех задач, запущенных предыдущим посетителем.
