Удаленный мониторинг

При подключении к GameClass под пользователем, включенным в группу Management, происходит автоматический переход в режим наблюдения (удаленного мониторинга). В этом режиме полностью отсутствуют функции оператора, но доступны следующие функции:
  • Настройка системы
  • Получение отчетов
  • Наблюдение за работой оператора в реальном времени
  • Наблюдение за работой оператора в прошедшем времени

Под наблюдением за работой оператора подразумевается то, что список компьютеров и их состояний в вашем GameClass будет выглядеть точно так же, как рабочее место оператора. Текущий отчет также будет отображать информацию на момент времени, который вы выберете.

Для перехода в нужную точку времени во время наблюдения пользуйтесь календарем и временем, для сброса времени на исходное - нажмите кнопку рядом с календарем.
