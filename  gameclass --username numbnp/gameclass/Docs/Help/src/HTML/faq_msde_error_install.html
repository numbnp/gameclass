Часто задаваемые вопросы

Во время инсталляции GameClass на фазе "Creating GameClass Database" происходит ошибка и инсталляция не удается, что делать?

В большинстве случаев проблема вызвана тем, что при удалении MSDE не удалились файлы базы данных GameClass (подразумевается что в начале вы удалили MSDE, а потом поставили его заново). Чтобы решить проблему, нужно вначале удалить файлы (а потом провести полную инсталляцию GameClass):
C:\Program Files\Microsoft SQL Server\MSSQL\Data\GameClass.mdf
D:\Program Files\Microsoft SQL Server\MSSQL\Data\GameClass_log.ldf
