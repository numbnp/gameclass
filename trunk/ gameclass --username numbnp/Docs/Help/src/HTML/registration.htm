Регистрация GameClass

Операцию регистрации необходимо проводить под учетной записью администратора Windows и под правами владельца клуба в системе GameClass. В окне «О программе» нажмите кнопку «Регистрация». В появившемся окне вставьте лицензионный ключ. Лицензионный ключ будет помещен в реестр и зашифрован с использованием серийного номера Вашего жесткого диска. Если Вы меняете жесткий диск Вашего компьютера, то необходимо повторить операцию регистрации. Ключ, скопированный из реестра кем-либо без Вашего согласия, не может быть использован на других компьютерах. Поэтому только Вы являетесь гарантом сохранности лицензионного ключа. Храните полученный лицензионный ключ в надежном месте. В случае, если Ваш лицензионный ключ будет обнаружен в сети интернет как доступный для общего пользования - он будет объявлен как украденный и анулируется. В случае потери Вами лицензионного ключа, Вам необходимо прислать запрос на info@gameclass.ru, напомнив нам Ваши регистрационные данные - на ранее указанные Вами e-mail адреса будет выслан регистрационный ключ.
